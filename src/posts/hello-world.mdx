# 블로그를 시작합니다

안녕하세요! 이 블로그는 **Next.js**와 **MDX**를 활용하여 만들었습니다.

## MDX란?

MDX는 마크다운 안에서 JSX를 사용할 수 있게 해주는 포맷입니다. 일반 마크다운의 간편함과 React 컴포넌트의 강력함을 동시에 누릴 수 있습니다.

## 코드 하이라이팅

코드 블록도 예쁘게 표시됩니다:

```typescript
const greeting = (name: string): string => {
  return `안녕하세요, ${name}님!`;
};

console.log(greeting("성우"));
```

인라인 코드도 가능합니다: `const x = 42`

## 리스트

- React 19
- Next.js 16
- Tailwind CSS 4
- MDX

> MDX로 블로그를 작성하면 마크다운의 편리함과 컴포넌트의 유연함을 모두 가질 수 있습니다.
